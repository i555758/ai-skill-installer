Install an AI Skill in One Click

Choose a use case below, download the installer and run it. Claude Code will be configured automatically.

Prerequisites: Claude Code CLI installed  •  Python 3 installed  •  HAI Proxy running (hai proxy)  •  SAP MCP Suite configured in ~/.claude/settings.json

How to install

1
Click "Download Installer" on the skill card below.
2
Double-click the downloaded .cmd file. No warnings.
3
The installer checks dependencies, creates the skill file and opens Claude Code automatically.
4
When Claude finishes, the terminal shows Concluido! — press Enter to close.
